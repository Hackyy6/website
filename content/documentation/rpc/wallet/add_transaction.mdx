Takes in a posted transaction, adds it to the wallet and mempool, and optionally broadcasts it to the network.

Returns the names of the wallet accounts involved in the transaction.

#### Request

```js
{
  transaction: string
  broadcast?: boolean
}
```

#### Response

```js
{
  accounts: string[]
}
```
