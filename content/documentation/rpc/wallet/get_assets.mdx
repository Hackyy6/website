## [wallet/getAssets]

Streams all the assets in the wallet of the given account. If not specified, the assets of default account will be returned.

#### Request

```js
{
  account?: string
  confirmations?: number
}
```

#### Response

```js
{
  createdTransactionHash: string;
  id: string;
  metadata: string;
  name: string;
  owner: string;
  status: string;
  supply: string | null;
}
```
