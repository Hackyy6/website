Gets an asset from the wallet from a given identifier and account

#### Request

```js
{
  account?: string
  confirmations?: number
  id: string
}
```

#### Response

```js
{
  createdTransactionHash: string
  creator: string
  id: string
  metadata: string
  name: string
  nonce: number
  status: string
  // Populated for assets the account owns
  supply: string | null
  verification: {
    status: 'verified' | 'unverified' | 'unknown';
  };
}
```
