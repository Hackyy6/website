## [wallet/create](https://github.com/iron-fish/ironfish/blob/master/ironfish/src/rpc/routes/wallet/create.ts)

Creates a new account in the wallet with the given name and optionally sets it as the default account.

#### Request

```js
{
  name: string
  default?: boolean
}
```

#### Response

```js
{
  name: string;
  publicAddress: string;
  isDefaultAccount: boolean;
}
```
