Streams block headers on gossip events

#### Request Body

```js
undefined
```

#### Response

```js
{
  blockHeader: {
    hash: string
    sequence: number
    previousBlockHash: string
    timestamp: number
    difficulty: string
    graffiti: string
  }
}
```
