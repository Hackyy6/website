## [rpc/getStatus](https://github.com/iron-fish/ironfish/blob/master/ironfish/src/rpc/routes/rpc/getStatus.ts)

Gets (and optionally streams) the status of the RPC server

#### Request

```js
{
  stream?: boolean
} | undefined
```

#### Response

```js
{
  started: boolean
  adapters: Array<{
    name: string
    inbound: number
    outbound: number
    readableBytes: number
    writableBytes: number
    readBytes: number
    writtenBytes: number
    clients: number
    pending: string[]
  }>
}
```
